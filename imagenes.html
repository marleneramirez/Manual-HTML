Imágenes



Sintaxis de imágenes HTML

La etiqueta HTML "img" se utiliza para incrustar una imagen en una página web. Técnicamente, las imágenes no se insertan en una página web; las imágenes están vinculadas a páginas web.

La etiqueta "img"crea un espacio de espera para la imagen a la que se hace referencia.

La etiqueta "img" está vacía, solo contiene atributos y no tiene una etiqueta de cierre.

La etiqueta "img" tiene dos atributos obligatorios:

  • SRC: especifica la ruta a la imagen
  • ALT: especifica un texto alternativo para la imagen
